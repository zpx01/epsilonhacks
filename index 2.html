A streamlined chatting experience for the user.

Using NLTK and Keras, SupplyChat is based on a true NLP framework.

SupplyChat learns from our curated JSON file filled with common medical supplies. Just input your desired product, and SupplyChat will tell you the quantity in an instant.

Advanced machine learning.

Our machine learning model is based on a a stochastic gradient descent with Nesterov accelerated gradient to generate the best responses.

Our training model for the chatbot is based on a 3-layer model: the first layer has 128 neurons, the second layer has 64 neurons, and the third output layer has the number of neurons equal to the number of intents to predict output intent with softmax. The number of times the model is trained is set at 20,000 to ensure maximum proficiency in the data.

Our chatbot GUI is based on the tkinter package and each visual representation of the chatbot has been backed by a Python function based on Keras.

Download the Python Script and Conda Environment !



The zip file includes the Python script, the JSON data file, and a Conda environment in a yml file.

Simply activate the conda environment (named chatbot) and run the Python script in your local machine's terminal. After the Python script completely runs, run the GUI and now, you will be able to use the chatbot!

Ensure that you are using Python 3.6 and have all necessary packages to run the chatbot script. If you face any issues, feel free to contact our team.
